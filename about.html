About Mood Waether

Mood Waether was born from the idea that your environment shapes your mood—and sometimes a little nudge helps you seize the day.

Key Goals

  • Provide accurate, up‐to‐the‐minute weather forecasts.
  • Offer personalized activity suggestions based on temperature trends.
  • Visualize weather data with interactive charts.

Tech Stack

  • HTML5 + semantic multi‐page structure
  • Responsive CSS with Flexbox and CSS Grid
  • JavaScript ES6 modules, classes, error handling
  • Open-Meteo free weather API
  • Chart.js for data visualization

Future Enhancements

Integrate user accounts, save favorite locations, and add hourly forecast charts.
